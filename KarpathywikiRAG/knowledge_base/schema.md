## YAML Frontmatter 規範

### 主題概念頁（必填欄位）

```yaml
---
title: 概念名稱（繁體）
type: concept
category: 天脈核心義理          # 見下方 category 枚舉
courses: L1 / L2 / L6          # 出現的課程，用 " / " 分隔
updated: YYYY-MM-DD
---
```

**category 枚舉（六選一，不得自創）**：

| 標籤 | 適用範圍 |
|------|---------|
| `天脈核心義理` | 天脈體系的核心修煉概念（任督/法身/覺知/業力等）|
| `L1核心框架` | L1 公開課的主要教學框架（三大秘密/五仙/三願八苦等）|
| `修煉功法` | 具體功法和修煉技術（河車/閉氣/靜功等）|
| `L6外部哲學` | L6 引用的外部思想家（拉玛那/Yachak/克里希那穆提等）|
| `L6商業交易` | L6 商業弘法、交易哲學主題（突發/IPO/資本家思維等）|
| `課程體系` | 天脈整體課程架構與事業系統 |

### 指針文件（必填欄位）

```yaml
---
title: 文件標題

version: V12
is_pointer: true
canonical_file: L1_知見_三願八苦與業力輪迴_V11.md
duplicate_group: D-005
重複組: D-005
重複狀態: 重複版本
重複性質: 遺漏更新
正本文件: L1_知見_三願八苦與業力輪迴_V11.md
---

> ⚠️ 此文件內容與正本完全相同（遺漏更新）。
> 正本請參見：[[L1_知見_三願八苦與業力輪迴_V11]]
> 如需更新本屆內容，請基於正本修改。
```

### 正本來源文件（主要欄位）

```yaml
---
title: 文件標題
course: L1
version: V11
category: 知見                 # 知見 / 功法 / 升班 / 儀軌 / 商業 / 修煉 / 交易
is_pointer: false
duplicate_group: D-005         # 若屬於重複組才填，否則留空
---
```

## 交叉引用規則

| 情況 | 格式 | 示例 |
|------|------|------|
| 引用概念頁 | `[[概念名]]` | `[[覺知力]]` |
| 引用其他概念頁（顯示名不同） | `[[文件名\|顯示名]]` | `[[任督二脉\|任督二脈]]` |
| 引用課程索引 | `[[LX_課程名/index\|顯示名]]` | `[[L1_公开课/index\|L1 公開課]]` |
| 引用正本文件（在指針中） | `[[文件名（不含.md）]]` | `[[L1_知見_三願八苦與業力輪迴_V11]]` |

**禁止**：在 wiki/ 頁面中寫絕對路徑（如 `/Users/bin/Documents/...`）

## 文件命名規範（raw/ 原始層）

```

示例：
L1_知見_覺醒生死觀人生目的_V2.md
L6_功法_河車自轉標準修煉_第05屆.md
L2_儀軌_連結文_V8.md
```
