## 文件命名規範

```
L{N}_{類別}_{主題}_{版本}.md

示例：
L1_知見_覺醒生死觀_V2.md
L2_功法_河車自轉_V8.md
L6_商業_資本家思維_第05屆.md
```

類別代碼：`知見 / 功法 / 升班 / 儀軌 / 商業 / 修煉 / 交易 / 素材`

---

## 怎麼把原始資料放進來

1. **整段未拆分的長文** → 先走 `模式 3 拆解課件`（讀 `../../拆分课件规则.md`），產出文件放到 `<根目錄>/4.资料/拆分课件/<課程名>/`，確認後再移到本目錄
2. **已經拆好的單篇文件** → 直接放到對應 `<課程>/<版本>/` 子目錄，命名遵循上方規範
3. 放好之後 → 走 `模式 4 Ingest` 編譯 wiki 層
